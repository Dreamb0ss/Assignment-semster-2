What is HTML?

HTML is used to create the structure of a webpage.

Common HTML Tags

  • <header> creates the top section.
  • <nav> creates navigation links.
  • <main> contains the main content.
  • <section> groups related content.
  • <footer> creates the bottom section.

What is CSS?

CSS is used to style and design webpages.

Why CSS is useful

  1. It adds colors.
  2. It controls layout.
  3. It makes websites look professional.
